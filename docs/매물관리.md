## 구현 완료 내역 (2026-03-03)

### 1. 사이드바 메뉴 변경
- "매물 관리" → "매물 데이터"로 변경
- "업로드한 매물 목록" → "매물목록"으로 변경
- "관심매물" 서브메뉴 추가 (`/real-estate/favorites`)
- "관리매물" 서브메뉴 추가 (`/real-estate/managed`)
- 파일: `src/components/layout/Sidebar.tsx`

### 2. DB 스키마 (ManagedProperty 모델)
- 매물 기본 정보: 매물명, 건물명, 주소
- 거래 유형: 매매, 전세, 월세
- 납부 일정:
  - 계약금 (금액 + 납부일)
  - 중도금 (금액 + 납부일)
  - 잔금 (금액 + 납부일)
- 계약 기간: 시작일, 만료일
- 금액 정보: 총 거래금액, 보증금, 월세
- 관리 정보: 세입자명/연락처, 책임자명/연락처, 메모
- 상태: active, expired, renewed
- 인덱스: userId, contractEndDate, finalPaymentDate, interimPaymentDate, status
- 파일: `prisma/schema.prisma`

### 3. API 엔드포인트
- `GET    /api/managed-properties` — 목록 조회 (재계약 필터, 납부일 필터, 상태 필터)
- `POST   /api/managed-properties` — 등록
- `PUT    /api/managed-properties/:id` — 수정
- `DELETE /api/managed-properties/:id` — 삭제
- 쿼리 파라미터: `renewalDays`, `paymentType`, `paymentDays`, `status`
- 파일: `src/server/index.ts`

### 4. 관심매물 페이지
- SavedProperty 테이블의 isFavorite=true 목록 표시
- 검색, 즐겨찾기 해제 기능
- 파일: `src/pages/real-estate/FavoritePropertyList.tsx`

### 5. 관리매물 페이지
- CRUD 폼 (모달): 기본정보, 계약기간, 납부일정, 금액, 담당정보, 메모
- 재계약 알림 필터: 전체 / 3개월 / 1개월 / 15일 / 7일 / 3일 / 1일 이내
- 상태 필터: 전체 / 활성 / 만료 / 갱신
- D-day 카운트다운: 만료일, 중도금 납부일, 잔금 납부일
- 긴급 시(7일 이내) 깜빡임 애니메이션
- 거래유형별 컬러 뱃지 (매매=빨강, 전세=초록, 월세=노랑)
- 파일: `src/pages/real-estate/ManagedPropertyList.tsx`

### 6. 라우트
- `/real-estate/favorites` → FavoritePropertyList
- `/real-estate/managed` → ManagedPropertyList
- 파일: `src/App.tsx`


### 7. 관심매물과 관리매물
- 매물목록 - 관심매물등록 -> 관심매물
- 매물목록 - 관리매물등록 -> 관리매물

- 관심매물 - 관리매물등록 -> 관리매물

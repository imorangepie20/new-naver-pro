# 공공데이터 연동 작업 명세

## 1) 목표
이 시스템은 특정 지역 부동산 매니저가 사용하는 실무 도구다.
핵심은 다음 3가지를 동시에 제공하는 것이다.

1. 전국 단위 부동산 동향
2. 내 사업권역(프로필 주소 기반) 중심 지역 분석
3. 내 보유 매물과 공공데이터를 비교한 실행 가능한 인사이트

결과는 표/차트/지도/시계열로 제공한다.

---

## 2) 산출물(필수)
- 전국 부동산 지표 대시보드
- 프로필 주소 기반 반경/행정구역 분석 대시보드
- 내 매물 대비 공공 시세/거래 이력 비교 리포트
- 시계열 변화(월/분기) 차트
- 이상치/기회 매물 탐지(예: 주변 거래가 대비 저평가)

---

## 3) 데이터 소스(1차 대상)
정확한 API 키/파라미터는 공공데이터포털 신청 후 확정한다.

### A. 국토교통부 실거래가
- 아파트 실거래
- 연립다세대 실거래
- 오피스텔 실거래
- 단독/다가구 실거래

활용 목적:
- 지역별 최근 거래가 분포(중앙값, 분위값)
- 동일/유사 면적 거래 추이
- 거래량 급증/급감 탐지

### B. 주소/행정구역 표준 데이터
- 법정동 코드, 행정동 코드 매핑

활용 목적:
- 프로필 주소 -> 분석 단위(읍면동/시군구) 정규화
- 네이버 매물의 cortarNo와 공공데이터 코드 정합

### C. (선택) 인구/가구/생활지표
- 통계청/KOSIS 등 공개 지표

활용 목적:
- 수요 측면 보조 지표(인구 증감, 가구수 변화)

---

## 4) 핵심 지표 정의

### 전국 지표
- 월별 거래량
- 월별 중위 거래가
- 거래 유형 비중(매매/전세/월세)
- 상위 상승/하락 지역 Top N

### 지역 지표(프로필 주소 기반)
- 최근 3/6/12개월 거래량 변화율
- 평균/중위 거래가 및 평당가(가능 시)
- 매물 타입별(아파트/오피스텔/빌라) 동향

### 내 매물 비교 지표
- 비교군 대비 가격편차(%): `(내 매물가 - 비교중위가) / 비교중위가`
- 유사 면적대 대비 편차
- 최근 거래일 기준 신선도
- 추천 액션:
  - 저평가 가능(매수/확보 후보)
  - 고평가 위험(가격 조정 검토)
  - 데이터 부족(추가 표본 필요)

---

## 5) 시스템 설계(프로젝트 기준)

### 5.1 수집 계층
- 배치 작업(일 1회 + 수동 실행)
- 원본 저장(Raw) + 정규화 테이블 분리
- 실패 재시도/부분 성공 허용

### 5.2 저장 계층
기존 `Property` 테이블은 유지하고, 공공데이터 전용 테이블 추가:
- `PublicTransactionRaw` (원본)
- `PublicTransaction` (정규화)
- `PublicAreaMonthlyStat` (월별 집계)
- `PropertyPublicComparison` (내 매물 비교 결과 캐시)

### 5.3 분석 계층
- 지역/기간 단위 집계 API
- 매물 비교 API
- 시계열 API

### 5.4 표시 계층
- 대시보드 카드 + 추이 차트 + 테이블 + 지도 오버레이

---

## 6) API 초안
기존 `/api` 네임스페이스에 추가한다.

- `POST /api/public/sync` : 공공데이터 동기화 실행(관리자)
- `GET /api/public/overview` : 전국 개요 지표
- `GET /api/public/region-trend?cortarNo=...&months=12` : 지역 시계열
- `GET /api/public/property-compare/:articleNo` : 내 매물 비교 결과
- `GET /api/public/opportunities?cortarNo=...` : 기회/주의 매물 목록

모든 API는 현재 인증 체계(`Authorization: Bearer`)를 따른다.

## 7) 화면 초안

### 7.1 전국 동향 탭
- KPI 카드: 거래량, 중위가, 증감률
- 월별 추이 라인차트
- 지역 상승/하락 랭킹 테이블

### 7.2 지역 분석 탭
- 프로필 주소 기준 기본 지역 자동 선택
- 반경/행정구역 전환
- 유형별 비교 차트

### 7.3 내 매물 비교 탭
- 내 매물 리스트 + 비교 결과 배지
- 상세 모달: 비교 근거 거래 목록, 편차, 추천 액션

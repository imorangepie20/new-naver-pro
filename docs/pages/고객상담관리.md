# 고객상담관리

## 고객상담관리
    - 고객정보관리 하단에 고객상담관리 메뉴 추가
    - 고객정보관리 게시판에 상담기록 버튼 추가하고 클릭하면 고객상담관리 페이지로 이동하여 입력할 수 있는 상태가 됨
## 고객상담기록
    - 고객정보를 기반으로 고객상담기록을 관리하는 페이지입니다.
        -- 고객명 입력
        -- 고객 전화번호 입력
        -- 고객 상담 내용 입력
        -- 고객명 입력시작 하면 고객명 과 전화번호 제시
        -- 제시된 고객명 선택할 수도 있고 직접 입력할 수도 있음
        -- 상담 내용 입력하고 저장
        -- 고객별로 상담 기록 그룹화
        -- 최신 상담 기록 부터 위로 정렬
        -- 상담 일시도 출력
        -- 상담 내용 수정 삭제 가능
        -- 고객정보에 상담기록이 추가됨
        -- 수정은 리드온리에서 더블클릭하면 수정가능하고 포커스 아웃되면 저장됨
        -- 삭제는 삭제 버튼 클릭하면 삭제됨

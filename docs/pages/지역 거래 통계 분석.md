# 지역 거래 통계 분석 http://imapplepie20.tplinkdns.com:5173/real-estate/address-market-stats
## 페이지 아래 내용으로 재 구현
## 지도 상단
    - 매물유형 선택 / 거래유형 선택
        -- 매물유형 : 아파트, 오피스텔, 상가, 토지, 주상복합, 단독다가구, 원룸, 다세대, 상업용부지, 공장창고, 호텔모텔, 기타
        -- 거래유형 : 매매, 전월세, 월세
    - 기본 선택은 아파트 / 매매
## 지도 마커 표시
    - 아파트
        -- 거래 데이터가 있는 아파트 단지 마커 표시
        -- 마커 클릭시 지도 하단에 아파트 거래 목록 출력
    - 그외
        -- 거래 데이터가 있는 매물 마커 표시
        -- 마커 클릭시 지도 하단에 거래 상세 정보 출력

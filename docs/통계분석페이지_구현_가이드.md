# 통계분석페이지 구현 가이드
## 0. 사전 작업
- 법정동형식의 주소필드 없는 경우
    -- 법정동형식의 주소필드 추가
    -- 법정동코드 필드 추가
    -- 법정동코드 필드에 법정동코드 입력
    -- 좌표 필드 추가하고 위도, 경도 입력
- 법정동형식의 주소필드 있는 경우
    -- 법정동형식의 주소 필드에 법정동 주소 입력
    -- 법정동코드 필드에 법정동코드 입력
    -- 좌표 필드에 위도, 경도 입력

## 1. 카카오 지도
- 카카오 지도 API를 사용하여 지도 표시
- 사무실 위치를 지도 중심점으로 설정
- 지도 중심점 주위 주요 부동산의 실 거래 정보를 수집하여 지도에 표시
- 지도에 표시된 부동산의 실 거래 정보를 클릭하면 해당 부동산의 실 거래 정보를 상세보기

## 2. 공공데이터 포털 이용
- 국토교통부 실거래가 API (공공데이터포털): 아파트, 연립다세대, 단독/다가구, 오피스텔, 토지, 분양/입주권, 상업업무용 부동산의 매매 및 전월세 거래 상세 정보를 제공합니다.
- 서비스키(인증키), 지역코드(5자리 법정동 코드), 조회월(YYYYMM), 페이지 번호 등을 URL 요청 변수로 사용.
- 응답 데이터에서 필요한 정보 추출하여 사용.
- 주의(2026-03-08 검증): 거래월 파라미터는 `DEAL_YMD`를 사용해야 실제 데이터가 조회됩니다.

- End Point: https://apis.data.go.kr/1613000/RTMSDataSvcAptTrade
